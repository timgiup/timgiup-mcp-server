# Categories — Danh mục bài đăng timgiup.com

API `/api/search` **chỉ hỗ trợ lọc theo 6 danh mục cha**. Dùng `slug` cho tham số `category`.

JSON: [`mcp_server/data/categories.json`](../mcp_server/data/categories.json)

| Slug | Tên |
|------|-----|
| `do-that-lac` | Đồ thất lạc |
| `thu-cung-that-lac` | Thú cưng thất lạc |
| `nguoi-than-that-lac` | Người thân thất lạc |
| `do-nhat-duoc` | Đồ nhặt được |
| `cong-dong-giup-do` | Cộng đồng giúp đỡ |
| `khac` | Khác |
